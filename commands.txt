1. git init - To initialize the repository
2. git repo add origin https://github.com/Harshil6327/Web-Developement_Git.git - Connect your local repository to remote repository 
3. git status - To check the status of the files if they are created or modified
4. git diff - To see the changes made to the files
4. git add filename - To add file to local git repository
5. git add . - To add multiple files to local git repository
6. git commit -am "added new files to repository" - This will commit all the changes made to the files
7. git push origin master - To push the changes to remote repository
8. git pull origin master - To pull the changes from the remoote repository
